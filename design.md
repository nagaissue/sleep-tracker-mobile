# Sleep Tracker App - デザイン計画

## アプリの目的
音声入力で睡眠記録（就寝、起床、昼寝）をGoogleカレンダーに自動記録し、睡眠統計を可視化するモバイルアプリ。

## 画面構成

### 1. ホーム画面（メイン）
**主要コンテンツ:**
- 大きなマイク入力ボタン（中央）
- 音声入力テキスト表示エリア
- 解析結果プレビュー（イベント一覧）
- 「カレンダーに登録」ボタン
- ステータスメッセージ表示

**ユーザーフロー:**
1. ユーザーがマイクボタンをタップ
2. 日本語で「昨夜11時に就寝して今朝6時に起床した」と話す
3. 音声認識結果がテキストで表示される
4. Gemini 2.5 Flash で自動解析
5. 「就寝 23:00」「起床 06:00」と表示
6. 「カレンダーに登録」をタップで Google Calendar に追加
7. 成功メッセージ表示

### 2. 統計ダッシュボード画面
**主要コンテンツ:**
- 過去7日間の平均睡眠時間
- 昼寝の回数
- 平均就寝時刻
- 平均起床時刻
- 日別睡眠時間グラフ

**ユーザーフロー:**
1. ホーム画面から「統計」タブをタップ
2. 過去7日間のデータを自動読み込み
3. グラフと数値で睡眠パターンを確認

### 3. 設定画面
**主要コンテンツ:**
- Google Calendar 連携状態表示
- Gemini API キー設定（オプション）
- アプリ情報

## 主要ユーザーフロー

### フロー1: 睡眠記録の追加
```
ホーム画面 → マイクボタンタップ → 音声入力 → テキスト表示 → 自動解析 
→ イベントプレビュー → 登録確認 → Google Calendar に追加 → 成功メッセージ
```

### フロー2: 統計確認
```
ホーム画面 → 統計タブ → データ読み込み → グラフ表示 → 睡眠パターン確認
```

## カラーパレット

| 用途 | 色 | 用途詳細 |
|------|-----|---------|
| 就寝イベント | 紺色 (#1e3a8a) | 夜間の落ち着いた雰囲気 |
| 起床イベント | オレンジ (#f97316) | 朝の活動的な雰囲気 |
| 昼寝イベント | 薄紫 (#a78bfa) | 休息の雰囲気 |
| プライマリ | 青 (#0a7ea4) | ボタン、アクセント |
| 背景 | 白/濃灰 (#ffffff / #151718) | ライト/ダークモード |
| テキスト | 黒/白 (#11181C / #ECEDEE) | 高コントラスト |

## UI コンポーネント

### マイクボタン
- 大きなサイズ（80x80px以上）
- タップ時のフィードバック（スケール、ハプティクス）
- 音声認識中は色が変わる
- 停止時は元の色に戻る

### イベントカード
- 時刻、イベント種別、カラーを表示
- 削除ボタン付き
- タップで詳細表示可能

### グラフ
- 日別睡眠時間を棒グラフで表示
- 過去7日間のデータ

## デザイン原則
- **シンプル**: 音声入力に集中できるシンプルなUI
- **視覚的フィードバック**: ユーザーの操作に即座に反応
- **ワンハンド操作**: 片手での操作を想定
- **アクセシビリティ**: 大きなボタン、高コントラスト
